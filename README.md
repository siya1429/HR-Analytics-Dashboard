 📊 Project 1: PhonePe Transaction Insights Dashboard

## 📌 Overview

This Power BI dashboard provides an interactive analysis of PhonePe transaction data, enabling users to monitor transaction performance, payment success rates, customer demographics, and service-wise transaction trends. The dashboard is designed to help businesses identify high-performing services, monitor payment status, and understand user behavior.

---

## 🎯 Objective

The objective of this project is to analyze digital payment transactions and provide meaningful business insights through interactive visualizations.

The dashboard helps answer questions such as:

* Which services generate the highest transaction value?
* How does transaction volume change over time?
* Which payment status has the highest percentage?
* Which age groups contribute the most transactions?
* How does weekday usage compare to weekend usage?

---

## 📈 Dashboard Features

* Total Transactions KPI
* Total Transaction Value KPI
* Unique Users KPI
* Payment Success Rate
* Monthly Transaction Trend Analysis
* Age Segment Contribution Analysis
* Service-wise Transaction Value Comparison
* Top 5 Users by Transaction Value
* Weekday vs Weekend Transaction Analysis
* Dynamic Filters (Month & Payment Status)
* Business Insights Panel

---

## 🛠 Tools & Technologies

* Power BI Desktop
* Power Query (Data Cleaning & Transformation)
* DAX (Calculated Measures & KPIs)
* Data Modeling
* Interactive Slicers
* Custom Visual Formatting

---

## 📊 Key Insights

* Loan services generated the highest transaction value.
* Successful transactions accounted for approximately 96% of all transactions.
* Users aged between 26–45 contributed the majority of transactions.
* Weekday transactions were significantly higher than weekend transactions.
* Monthly transaction values remained relatively stable with seasonal fluctuations.
